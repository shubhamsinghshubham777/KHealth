KHealth

expect class KHealth

KHealth is a Kotlin Multiplatform wrapper around Android's Health Connect and Apple's HealthKit APIs. It provides a set of convenient properties and methods to perform most common operations easily (like checking/requesting permissions, reading/writing data, and more).

It is an expect class whose instance can normally be created without needing to pass any parameters on Apple platforms, but on Android, it requires the reference to a ComponentActivity to be able to subscribe to Health Connect permission events and detect whether the user has granted a permission or not. Hence, its instance needs to be created separately on different platforms and then passed on to the common/shared code.
